# UC-1.2 登入/登出

| 項目 | 內容 |
|------|------|
| **填表人** | 李芝瑢 |
| **商業流程編號** | UC-1.2 |
| **行為者** | 訪客 / 會員 |
| **內容概述** | 驗證使用者身份進入系統或登出。 |
| **先決條件** | 使用者已完成註冊。 |
| **後置條件** | 成功登入後導向首頁,登出後返回登入頁。 |
| **主要流程** | 1. 輸入帳號密碼。<br>2. 系統驗證。<br>3. 登入成功顯示個人化頁面。<br>4. 登出時清除 session。 |
| **替代流程** | a. 密碼錯誤  提示重新輸入。<br>b. 連續錯誤3次  鎖定5分鐘。 |
| **輔助說明** | 希望能不要紀錄輸入的帳密,除非要設定。 |
| **Screen 畫面** | 登入頁、首頁 |
