# UC-1.4 刪除帳號

| 項目 | 內容 |
|------|------|
| **填表人** | 李芝瑢 |
| **商業流程編號** | UC-1.4 |
| **行為者** | 會員 |
| **內容概述** | 使用者可刪除個人帳號以保障隱私。 |
| **先決條件** | 使用者已登入且確認刪除動作。 |
| **後置條件** | 帳號與資料永久刪除。 |
| **主要流程** | 1. 使用者按下「刪除帳號」。<br>2. 系統顯示警告。<br>3. 使用者再次確認。<br>4. 系統刪除資料並登出。 |
| **替代流程** | a. 若使用者取消刪除  返回設定頁。 |
| **輔助說明** | 顯示「刪除前提示」與「備份選項」,遵守 GDPR / 個資法規。 |
| **Screen 畫面** | 帳號設定頁 |
